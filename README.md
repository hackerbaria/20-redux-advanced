## 🛠️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/hackerbaria/20-redux-advanced.git
cd 20-redux-advanced
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the Development Server

```bash
npm start
# or
yarn start
```
